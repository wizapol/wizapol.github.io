Proxmox automated scripts by wizapol

Firefly III es una plataforma de código abierto para la gestión de finanzas personales. Ofrece contabilidad de partida doble, importación de datos, reglas avanzadas para organizar transacciones, y opciones para etiquetar y presupuestar gastos. También cuenta con informes detallados y una API REST JSON para usuarios avanzados. Es una solución completa y fácil de usar para manejar tus finanzas.

Sistema Operativo: Alpine 3.18.

Configuración de VM: 1 CPU, 1GB RAM, 1GB Storage.

Corre este comando en la consola de Proxmox:

bash -c "$(wget -qLO - https://raw.githubusercontent.com/wizapol/Proxmox-scripts/main/fireflyiii/fireflyiii.sh)"

Para este aplicativo, ingresar a: "(IP_VM):8200"
