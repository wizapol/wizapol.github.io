FireFly III

Firefly III es una plataforma de código abierto para la gestión de finanzas personales.

Ofrece contabilidad de partida doble, importación de datos, reglas avanzadas para organizar transacciones, y opciones para etiquetar y presupuestar gastos.

También cuenta con informes detallados y una API REST JSON para usuarios avanzados. Es una solución completa y fácil de usar para manejar tus finanzas.


Sistema Operativo: Alpine 3.18.

Configuración de VM: 1 CPU, 1GB RAM, 1GB Storage.

Corre este comando en la consola de Proxmox:

bash -c "$(wget -qLO - https://raw.githubusercontent.com/wizapol/Proxmox-scripts/main/fireflyiii/fireflyiii.sh)"

Para este aplicativo, ingresar a: "(IP_VM):8200"

Ubuntu 23.04

Este script instala Ubuntu 23.04 en un contenedor de Proxmox. Ideal para entornos de desarrollo, pruebas o producción.

Sistema Operativo: Ubuntu 23.04.

Configuración de VM: 1 CPU, 1GB RAM, 4GB Storage.

Corre este comando en la consola de Proxmox:

bash -c "$(wget -qLO - https://raw.githubusercontent.com/wizapol/Proxmox-scripts/main//ubuntu23/ubuntu23.sh)"

Para acceder a este contenedor, usar SSH o la consola de Proxmox y conectar a la IP asignada.
